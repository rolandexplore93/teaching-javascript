When using media query for Responsiveness, you use max-width & min-width

Breakpoint for different screen sizes

  • Extra small screen size (mobile): < 360px
  • small screen size (mobile): => 360px
  • medium size (large phones): => 576px
  • medium 2 (tablets): => 768px
  • Large (laptops): => 992px
  • x-large (laptop, small screen desktop): => 1200px
  • xx-large (desktops, large laptops): => 1440px
